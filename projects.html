01

This Website

HTML CSS GitHub

Creating this website was a project in itself, centered around developing and utilizing my skills with HTML, CSS, and GitHub. The goal was to create a clean but professional profile that could highlight information about myself and my career. I also wanted to ensure it was optimized for mobile devices so that I could display it easily to others. Since this was my first major project using these web development tools, I had my fair share of roadblocks. The main issue I faced came from a difficulty with a remote repository on GitHub, where the main branch was corrupted or had broken references. While this was initially intimidating, I researched the problem and, with some perseverance, learned how to create a temporary branch to preserve my changes, delete the problematic main branch, and then recreate it to restore the repository's integrity. This experience tremendously grew my confidence using the terminal and my ability to use GitHub as a whole. I plan to continue refining this site as I hone my capabilities.

02

Yelp Restaurant Rating Prediction Model

GitHub
Python Machine Learning Linear Regression Data Science

In this project, I developed a machine learning model to predict Yelp ratings for restaurants based on features such as review sentiment, price range, amenities, and more. Using a dataset from Yelp that includes business attributes and review data, I performed data cleaning, preprocessing, and exploratory analysis to identify key factors influencing restaurant ratings. I trained a multiple linear regression model, evaluated its performance with R² scores, and fine tuned it for optimal prediction accuracy. This project demonstrates my ability to process and analyze real world data, build and evaluate machine learning models, and interpret model results to derive actionable insights.

03

Machine Learning Movie Recommendation System

GitHub
PyTorch Deep Learning Python Neural Networks

In this project, I built a personalized movie recommendation system using a neural network implemented with PyTorch. The model learns from user rating data to identify patterns in viewing preferences and generate tailored movie suggestions. This involved preprocessing and structuring the dataset, designing and training a neural network architecture, and evaluating recommendation quality to refine the model's accuracy. The project gave me hands on experience applying deep learning to a classic real world problem in data science, building systems that can meaningfully personalize content for users. It also deepened my understanding of PyTorch workflows, including model design, loss functions, and iterative training.

04

Containerized Ride Share Application

GitHub
Python Flask Docker DevOps SQL

This project is a ride share web application built with Python and Flask, containerized using Docker to ensure a consistent and portable deployment environment. The application supports core ride share functionality and uses a database for persistent storage of user and ride data. Containerizing the app with Docker was a key focus of the project, giving me practical experience with building and managing Docker images, configuring containers, and understanding how containerization streamlines the development and deployment workflow. This project strengthened my backend development skills and introduced me to the fundamentals of DevOps practices, particularly around packaging and deploying applications in isolated, reproducible environments.
