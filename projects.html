Yelp Restaurant Rating Prediction Model

In this project, I developed a machine learning model to predict Yelp ratings for restaurants based on features such as review sentiment, price range, amenities, and more. Using a dataset from Yelp that includes business attributes and review data, I performed data cleaning, preprocessing, and exploratory analysis to identify key factors influencing restaurant ratings. I trained a multiple linear regression model, evaluated its performance with R² scores, and fine-tuned it for optimal prediction accuracy. The goal of this project was to apply data science and machine learning techniques to solve a real-world problem: understanding the factors that impact a restaurant's success based on customer feedback. This project demonstrates my ability to: Process and analyze real-world data, Build and evaluate machine learning models, and Interpret model results to derive actionable insights. Containerized Ride Share Application

This project is a ride share web application built with Python and Flask, containerized using Docker to ensure a consistent and portable deployment environment. The application supports core ride share functionality and uses a database for persistent storage of user and ride data. Containerizing the app with Docker was a key focus of the project, giving me practical experience with building and managing Docker images, configuring containers, and understanding how containerization streamlines the development and deployment workflow. This project strengthened my backend development skills and introduced me to the fundamentals of DevOps practices, particularly around packaging and deploying applications in isolated, reproducible environments.
